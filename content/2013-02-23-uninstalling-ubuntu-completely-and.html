--- title: "Uninstalling Ubuntu completely and restoring windows boot loader" layout: "post" date: "2013-02-23 16:55:00" updated: "2013-02-23 17:00:31" categories: [tech] tags: [windows 7, grub, windows, ubuntu] --- Windows users install Linux alongside with windows. When they want to uninstall Ubuntu, they are likely to remove grub boot loader and make their windows operating system not bootable. Ubuntu installs grub loader which overrides windows boot loader. So before uninstalling Ubuntu from the system, we need to reset master boot records to point windows boot leader. Before attempting this, please make sure that you took backup of important files and folders from your Ubuntu OS. Reboot the computer and select windows 7 from the grub boot list. Now immediately press F8. This will list the available boot options for windows. From this list select >Repair Your Computer. This will load a new window and will ask your administrative account. Give the user name and password. Select Command Prompt in the window as shown the below picture. []

  • Now type the following commands in command prompt.
    • BootRec /FixBoot
    • BootRec /FixMbr
  • This commands will restore your windows boot loader. FixBoot will write a new boot sector onto the system partition (C:\). FixMbr writes the master boot record of the system partition.
  • Now exit the command prompt and press Restart button to restart the computer. Now the computer will directly boot into Windows. Thats it. 
  • Now we have to recover the space assigned to Ubuntu installation. To do this, Go to Control Panel\All Control Panel Items\Administrative Tools\Computer Management and then click disk management.
  • Right click the Linux partition which was assigned to Ubuntu and delete it. 

  • Now right click on the free space of Disk 0 and select "New Simple Volume...". Assign a new drive letter. That is it. This volume will be available as a new partition in your computer.
