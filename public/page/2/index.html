Python List Comprehensions and generator expressions
  • March 19, 2017
  • Python  
Comprehensions are one of the prominent features of Python. Once you understand them, it helps avoiding lots of redundant and repetitive code. It helps creating lists, dictionaries and generators from other iterable objects with a single statement. List Comprehensions Let’s take the below tuple as an example, scores = ( ('Science', 85), ('Phisics', 65), ('Mathematics', 89), ('Computer Science', 34) ) To get all the subjects from this tuple, >>> s = [subject for subject, score in scores] >>> s ['Science', 'Phisics', 'Mathematics', 'Computer Science'] The above statement with square bracket is called list comprehensions.
